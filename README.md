# Criterion Onboarding

A Next.js onboarding experience built with React, Tailwind CSS, Framer Motion, and Three.js.

## Getting Started

Install dependencies and run the development server:

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

## Tech Stack

- **Next.js 16** – React framework
- **Tailwind CSS** – Styling
- **Framer Motion** – Animations
- **Three.js / React Three Fiber** – 3D graphics
- **Lucide React** – Icons

## Scripts

| Command | Description |
|---------|-------------|
| `npm run dev` | Start development server |
| `npm run build` | Build for production |
| `npm run start` | Start production server |
| `npm run lint` | Run ESLint |

## Deployment

This app is configured for **Firebase Hosting**. Deploy with:

```bash
npm run deploy
```

**Live site:** [https://criterion-onboarding.web.app](https://criterion-onboarding.web.app)

See [FIREBASE_SETUP.md](./FIREBASE_SETUP.md) for first-time Firebase setup.
